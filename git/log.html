log

git log默认会输出commit hash, author, date, commit message

  • --oneline 仅仅输出commit hash 前7个字符串和commit message

git log --oneline

f6a3d03a41 (HEAD -> 4.0.0-RC2-release, origin/dev) fix g4 for assignmentValueNode get error by index (#3290)
247c9182ce Add sharding + encrypt test engine (#3298)
bd885b8cbe Add table owner for logic encrypt column (#3276)
763b02ff56 Add OrderByContextEngineTest (#3293)
  • --stat 输出文件增删改的统计数据
  • -p 控制输出每个commit具体修改的内容
  • shortlog 命令用来输出汇总信息，以作者进行分类
  • .....
