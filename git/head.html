HEAD引用

HEAD文件是一个符号引用，指向目前所在的分支。该文件存放在.git/目录下

所谓符号引用，意味着它并不像普通引用那样包含一个SHA-1值,它是一个指向其他引用的指针。

如：当前分支是master 那么HEAD中存放的就是引用对象master文件的地址

cat .git/HEAD

ref: refs/heads/master

更新HEAD文件也就是切换分支，将其更新为指定分支的地址 如：

git checkout dev
Switched to branch 'dev'

cat .git/HEAD
ref: refs/heads/dev
