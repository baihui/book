常量池

它是方法区的一部分，方法区存放了类的信息描述符号，对类内部定义属性，值的描述是以及调用其他类的信息是存储在常量池中，通过解析阶段将符号引用替换成直接引用，作为该类访问外部类的入口或者调用自身信息的入口 如变量名称，值，引用，引用的信息以及也可以在运行期间添加新的常量放入池中，String类的intern()方法。

以下类型表示该类需要调用的其他类的属性以及方法或者自身的

  • Class（类） 引用其他类或者自身

  • Fieldref（字段） 引用其他类或者自身的

  • Methodref（方法）

    只有用到才会出现在常量池中包括自身的方法

  • NameAndType（方法的参数类型或者字段的）

public class A {
    public final int anInt = 100 ;
    public final String anString = "abc" ;
    public int bnInt = 99 ;
    public final double bnDoble = 66.8 ;
    public final float bnFloat = 55 ;
    public final Integer bnInteger = 24 ;
    public  Integer bInteger = 20 ;
    public void aVoid(){

        int i = 0 ;
        int a = i++ ;
        a = --i ;
        a = ++i ;
        a = i-- ;

        i++ ;
        System.out.println(i);
    }
}
  • 符号引用
如下索引1通过符号来表示引用的对象以及方法：
#1 = Methodref #17.#44 // java/lang/Object."<init>":()V
#2 = Fieldref #16.#45 // com/bh/Application/A.anInt:I
#2表示常量池中索引值：指向的类型是Fieldref表（字段表）：#16（java语言类型）.#45（引用的方法或者其他） 
#16 = Class #59 // com/bh/Application/A
#45 = NameAndType #18:#19 // anInt:I
NameAndType 表示定义的名称和类型(java)
#18 = Utf8 anInt (uft8编码的字符串名称)
#19 = Utf8 I （表示基本数据类型int）

#21 = Integer 100 //字面量
#9 = Float 55.0f
#6 = Double 66.8d

被final修饰的基本数据类型和字符串存放在其中，但是Integer对象型即使被修饰也不会存储中


public class B {
    //a的值会被存放在常量池中以及字段的定义中
    public final int a = 10 ;
    //是通过类的构造函数予值
    public final Integer b = 20 ;
    //通过static 块予值
    public static int c = 30 ;
}

  • 被final修饰的基本类型

    常量池中索引10的位置存放了a的值同时在字段定义中也存放了值

  • 包装类

    没有将值存放在常量池中，而是通过类的构造函数给予值

被final修饰的基本类型值会存放在常量池中，而基本数据类型的包装类即是final static 修饰也不会存放在常量池中他们的值都是通过类的构造函数或者静态块给予值的,而String类型比较特殊的不会被final修饰也全部存放

常量池中存放的符号引用一般是在两个阶段进行修改 解析阶段和准备阶段一种是指向直接引用另外一种是分配默认值或者真正值，
