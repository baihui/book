反射

反射机制定义

  • 对于任意一个类，通过该类的Class都能够直到这个类的所有属性和方法以及创建该类的实例。

  • 对于任意一个对象，通过该类的Class都能够返回并且调用Method（invoke）或者设置Field（Set）它的任意一个方法和属性。

Class类和java.lang.reflect类库一起构成了对Java反射机制的支持。其中最常使用到的类是Constructor，Field，Method，而这三个类都继承了一个接口java.lang.reflect.Member

  • Constructor
    • 提供关于类的单个构造方法的信息以及对它的访问权限。
  • Field
    • 提供有关类或接口的单个字段的信息，以及对它的动态访问权限，反射的字段可能是一个类（静态）字段或实例字段。
  • Method
    • 提供关于类或接口上单独某个方法访问标志（Modifier ）的信息。所反映的方法可能是类方法或实例方法（包括抽象方法）。
  • Modifier
    • 访问标志权限 类提供了 static 方法和常量，对类和成员访问修饰符进行解码。修饰符集被表示为整数，用不同的位位置 (bit position) 表示不同的修饰符。该类的字段均是int类型的变量。
  • ReflectPermission
    • 反射操作的 Permission 类。ReflectPermission 是一种指定权限，没有动作。当前定义的唯一名称是 suppressAccessChecks，它允许取消由反射对象在其使用点上执行的标准 Java 语言访问检查 - 对于 public、default（包）访问、protected、private 成员。
