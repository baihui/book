静态类

如果一个类要被声明为static的，只有一种情况，就是静态内部类。如果在外部类声明为static，程序会编译都不会过

  • 静态类和静态属性解析一样，也是其在类加载期间确定类型，属于静态解析

  • 静态内部类跟静态方法一样，只能访问静态的成员变量和方法（不会引发外部类的加载），不能访问非静态的方法和属性，但是普通内部类可以访问任意外部类的成员变量和方法.

  • 静态内部类可以声明普通成员变量和方法（内部类必须通过外部类的对象创建），而普通内部类不能声明static成员变量和方法。

静态内部了不会引发外部类的加载，只是定义了不同名称的类

public class B  {

    static {
        System.out.println("B........");
    }

    public static class BB{
        static {
            System.out.println("bb........");
        }
    }

A类中调用

public static void main(String as[]){

    //不会引发BB类的加载
    // B b = new B() ;

    //======== 也不会引发外部类的加载，只有内部类是static类才可以不用创建外部类直接访问
    B.BB bb = new B.BB() ;

     //例如现在BB 不是内部类类，那么必须 先创建外部类，在通过外部了的创建内部类，类似一个外部类的普通属性
        B b =   new B() ;
        B.BB bb =  b.new BB() ;
}

内部类只是定义了名称为B$BB类名，所以除了可以static 后继承 其他的和普通类无任何区别，以及调用外部类,类似于静态属性，但是它不会引发外部类的加载
