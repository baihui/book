私有仓库

私有仓库-官方仓库中提供了Registry的镜像

仓库数据路径

默认情况下，注册表数据将作为docker卷保留在主机文件系统上 主机目录绑定安装~/registry到注册表容器中的/var/lib/registry/

sudo run -d -p 5001:5001 --restart=always --name registry registry:2
