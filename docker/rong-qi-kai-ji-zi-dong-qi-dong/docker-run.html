docker run

运行一个新的容器

run退出状态码如下：

  • 0，表示正常退出
  • 非0，表示异常退出（退出状态码采用chroot标准）
    • 125，Docker守护进程本身的错误
    • 126，容器启动后，要执行的默认命令无法调用
    • 127，容器启动后，要执行的默认命令不存在

其他命令状态码，容器启动后正常执行命令，退出命令时该命令的返回状态码作为容器的退出状态码
